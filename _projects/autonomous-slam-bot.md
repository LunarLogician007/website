- Simulated an **Autonomous Robot** in a custom Gazebo world using Robot Operating System (ROS2) and **SLAM Toolbox**

- Interface a **Raspberry Pi 5** with **ESP32** via **UART** to transmit ROS2 commands to stepper motors for precise control
